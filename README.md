# Profile Manager

A profile management system for vscode.

## Features

* Create Extension Profiles
* Ignore Extensions

## Extension Settings

* `profileManager.profiles`: profile list
* `profileManager.ignored`: list of ignored extensions
* `profileManager.currentProfile`: current profile
